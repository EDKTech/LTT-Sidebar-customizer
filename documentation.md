# Documentation
------------------------------
#### This documentation will list details about all the options provided by LTT Forum Customizer.
------------------------------

## Sidebar

#### Hide Ad - Hides the ad on the top of the sidebar.
![image](https://user-images.githubusercontent.com/84492239/169652016-3b9009a8-cb10-4d1a-a2f9-7ad1ac6c6b8c.png)


#### Hide Recent Topics - Hides the Recent Topics Section on the sidebar.
![image](https://user-images.githubusercontent.com/84492239/169652025-5693f23e-f35b-41a2-9e3f-acd3202dbc24.png)


#### Hide Featured Topics - Hides the Featured Topics section.
##### (No screenshot as there are no featured topics for me to screenshot at the moment)

#### Hide LMG Social Links - Hides LMG Social links section(Discord, Merch, Floatplane, etc)
![image](https://user-images.githubusercontent.com/84492239/169652101-424affdc-a3c3-4e26-93b9-aa64615715c6.png)

#### Hide Recent LMG Videos - Hides the latest videos from LMG on the sidebar. You can either select this option to select/deselect all, or choose the ones which you'd like to show.
![image](https://user-images.githubusercontent.com/84492239/169652179-6a93001b-380a-4c7b-bd02-287ac0f8902a.png)

#### Show Status Updates on all pages - Shows Status Updates section on all pages, instead of the Forum default of the main page only.
![image](https://user-images.githubusercontent.com/84492239/169652234-ad527da4-81e0-4919-b47e-a5cef5b24005.png)


## Profiles

#### Hide Rank icons
![image](https://user-images.githubusercontent.com/84492239/169652484-fbacb5cd-8e25-43c8-bb0c-32c559332ff7.png)
![image](https://user-images.githubusercontent.com/84492239/169652510-fc416235-2599-40ac-9e25-ff38d1c3b2b1.png)

#### Replace the "Member" title with the user's rank - This option hides the group name for "Members" and replaces it with the user's rank. All other groups such as Moderator, Administrator, etc remain as is.
![image](https://user-images.githubusercontent.com/84492239/169652641-ed14c6e6-525f-4250-b78a-bdbce0f32d34.png)
![image](https://user-images.githubusercontent.com/84492239/169652653-449ae78d-43d4-4ed2-9286-84cb2b8c20fa.png)

#### Replace Post count with complete post count - By default, the forum does not show the full post count, instead a rounded off figure. Example, 2.2k instead of 2156. The icon shown before the number is dependent on the next option, Remove The Solutions Count. If this option is selected, the post count will not be shown with the icon, but instead "xxx posts".
![image](https://user-images.githubusercontent.com/84492239/169652748-91d4f9b9-8b04-485f-9ac7-5129a0d3f7e3.png)
![image](https://user-images.githubusercontent.com/84492239/169652760-00919983-bf28-4672-a4ef-2baab1c95d72.png)

#### Remove the Solutions Count
![image](https://user-images.githubusercontent.com/84492239/169652781-27a3b2f2-f909-4be2-b566-646e67448c75.png)

#### Hide Member Group - Hides the group for users that already have badges signifying it. This is hidden for Administrators, Moderators, LMG Staff and Bronze, Silver and Gold Contributors.
![image](https://user-images.githubusercontent.com/84492239/169653501-32d3fff8-8dc3-450d-8775-b4e7ea13c63f.png)
![image](https://user-images.githubusercontent.com/84492239/169652883-b4929d8b-7be5-4d78-8071-b01ddedcd702.png)

#### Show location below Member's profiles - Shows the location of a member(if they have set one on their profile, of course) under their profile.
![image](https://user-images.githubusercontent.com/84492239/169652938-e63def0a-0a4a-48cc-b191-14893af0029a.png)

#### Show your rank in profile dropdown - Shows your rank stats in the profile dropdown. This, for some reason is only shown on the mobile version of the page.
![image](https://user-images.githubusercontent.com/84492239/169653005-2fbb58ed-d124-40f4-bbf5-bacaba6eb2e2.png)

#### Set badge margin to 3px - Sets the margin for badges(Moderator, Administrators, etc) to 3px. This is set to 8px(top) by default by the forum.
![image](https://user-images.githubusercontent.com/84492239/169653080-86bcbcca-9560-40c2-b14b-9fdee8d7b407.png)
![image](https://user-images.githubusercontent.com/84492239/169653099-ee972e7b-dff4-4394-bb9e-0c2be408fe40.png)
